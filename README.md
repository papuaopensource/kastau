# Kastau

Kastau adalah platform kebijakan publik Kabupaten Jayapura — menyajikan peraturan daerah, peraturan bupati, dan SK Bupati dalam bahasa yang mudah dipahami oleh warga.

**Web:** [kastau.pages.dev](https://kastau.pages.dev)

## Fitur

- Pencarian kebijakan berdasarkan kata kunci, topik, atau nomor peraturan
- Kategori kebijakan: infrastruktur, pendidikan, kesehatan, lingkungan, UMKM, dan lainnya
- Ringkasan kebijakan dalam bahasa yang ringkas dan mudah dipahami
- Status kebijakan: berlaku, direvisi, atau dicabut

## Tech Stack

- [Astro](https://astro.build) — static site framework
- [Tailwind CSS](https://tailwindcss.com) — styling
- [FastAPI](https://fastapi.tiangolo.com) — backend API
- [Turborepo](https://turbo.build) — monorepo build system
